#### *Infrastructural-Layer:-*
    - The Lowest Technical Layer.
    - Physical and Technical Foundation.

*Properties:-*
    - Provides the Hardware and Platform on which ERP runs.

*Includes in Infrastructural Layer:-*
    - Servers
    - Networks
    - Operating System
    - Cloud Platform
    - Security System

*Components:-*
    - Application Servers.
    - Database Servers.
    - Storage.
    - Firewalls.
    - Load Balancers.

*Deployment-Options:-*
    - On-Premise Servers.
    - Cloud (AWS,Azure,GCP).
    - Hybrid-Infrastructure.

*Responsibilities:-*
    - Performance
    - Availability
    - Scalability
    - Security
    - Disaster Recovery

*Tech-Stack:-*
    - Linux/Windows Server
    - Docker
    - Kubernates
    - Virtual Machines
    - Networking
