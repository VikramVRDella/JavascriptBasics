### Layers of ERP (Enterprise Resource Planning):-
    - User Interface Layer (Handles UI)
    - Application / Business Layer Logic (Handles Request from Frontend to Backend)
    - Integration Layer (Handles the Payment and other External ERP Modules)
    - Data Layer (Handles the storage of Data in Databases)
    - Infrastructure Layer (Allocates the Infrastructure of all those layers)

*Flow of the Layers:-*
`User Interface ----> APIs -----> Application Server -----> Business Modules -----> Services -----> Workflow Engines -----> Database`
