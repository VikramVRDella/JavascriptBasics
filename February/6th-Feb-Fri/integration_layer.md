#### *Integration-Layer:-*
    - It is a Communication Layer.
    - It connects ERP with other Systems.

*Properties:-*
    - Allow Different Modules to talk.
    - Connect External Applications.
    - Enable Data Exchange.

*Need for Integration Layer:-*
    - Banks.
    - Payment Gateways.
    - Legacy Systems.
    - Partner Systems.
    - E-Commerce Platform.

*Includes on Integration-Layer:-*
    - API.
    - Web Services.
    - Message Queries.
    - Middleware.
    - Connectors.

*Scenarios:-*
    - Sales ----> Inventory -----> Finance
    - These three are Interconnected.
